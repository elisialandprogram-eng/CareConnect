GOLDENLIFE ADMIN PROVIDER COMMAND CENTER REDESIGN

OBJECTIVE

Redesign the Admin → Providers section into a true Provider Command Center.

This is a UI/UX and information architecture redesign.

IMPORTANT:

DO NOT create new provider APIs unless absolutely necessary.

FIRST reuse and compose existing APIs, queries, hooks, DTOs, and data already available in the platform.

The goal is to surface existing information in a better structure with better visibility and admin efficiency.
